Ques. Explain inline , internal and external stylesheet

Ans. In web development, there are three main ways to apply CSS (Cascading Style Sheets) to HTML: inline, internal, and external stylesheets.
1. Inline Stylesheet CSS is applied directly to the HTML element using the style attribute. Only affects the specific element. Useful for quick, one-off styles but not recommended for larger projects .
2. Internal Stylesheet CSS is placed within the style tag inside the head section of the HTML document. Styles apply only to that specific page.
3. External Stylesheet CSS is written in a separate .css file. The CSS file is linked to the HTML using the link tag in the head.

Ques. Justify-content allows you to do what?

Ans. The justify-content property in CSS Flexbox and Grid layout is used to align items along the main axis of a container. The main axis is horizontal by default (flex-direction: row) but can be vertical if flex-direction: column is used. It controls how space is distributed between items and where items are positioned in the container.

Ques. Difference between absolute and relative positioning

Ans. relative :
The element is positioned relative to its original position in the document flow. It remains in the document flow, so other elements are not affected. You can use top, right, bottom, and left to move the element relative to its starting position.


absolute:
The element is removed from the normal document flow, meaning it doesn't affect other elements. It is positioned relative to the nearest positioned ancestor (relative, absolute, fixed, or sticky). If no positioned ancestor exists, it positions itself relative to the (viewport).

Ques. What is the z-index in CSS?

Ans. The z-index in CSS is a property that controls the stacking order of positioned elements along the z-axis (the imaginary depth axis). It determines which elements appear in front of or behind others when they overlap.
Positive integers: Brings elements forward (z-index: 10).
Negative integers: Sends elements backward (z-index: -1).
Auto: Uses the default order (z-index: auto).

Ques. What is the difference between padding and margin?

Ans. In CSS, padding and margin are both used to create space around elements, but they work in different ways:
1. Padding :
Adds space inside an element, between the element's content and its border. Expands the element's size, affecting its background color and border area.
